## Features

- AES-GCM encryption fully in the browser
- Secure sharing of notes via encrypted URL hash
- Markdown editor with live preview
- Dark / Light mode support (automatic + manual toggle)
- Works entirely on GitHub Pages (static site)
- No data stored on any server ‚Äî privacy first

---

## How to use

1. Write your notes in the Markdown editor.
2. Enter a password to encrypt your note.
3. Click **Generate link**.
4. Share the generated URL with your friend.
5. Your friend opens the link and enters the password to decrypt and view the note.
